# UAFX_Ruby'63
1963年製のトップ・ブースト回路が組み込まれた VOX AC30  
1961年製のトップ・ブースト回路が入っていない VOX AC30  
をモデリング。  
  
## 1.設定のしかた
1. 「ALT/AMP/STORE」トグルスイッチはAMPに、ノブはすべて12時にしておく。
2. 「MOD」トグルスイッチで、アンプを３つのなかから選ぶ。
3. 「SPEAKER」トグルスイッチで、スピーカー・キャビネットを６つのなかから選ぶ。
4. 音を出しながら、「OUTPUT」ノブで、出力レベルを調整する。
5. 「VOLUME」ノブで、ゲインを調整する。一度、ゼロ・9時・12時・3時・フルなどを試すこと。
6. 「BOOST」ノブで、ブースト量を調整する。一度、ゼロ・9時・12時・3時・フルなどを試すこと。
7. 「BASS」ノブで、低域特性を調整する。
8. 「TREBLE」ノブで、高域特性を調整する。
9.  「REVERB」ノブで、スプリングリバーブの掛かり量を調整する。
10. 「ALT/AMP/STORE」トグルスイッチをALTにして、リバーブの設定を調整する。
11. 「SPEED」ノブで、ビブラート・スピードを調整する。一度、ゼロ・12時・フルなどを試すこと。
12. 「INTENSITY」ノブで、ビブラートの強さを調整する。一度、ゼロ・12時・フルなどを試すこと。

## 7.設定メモ
まずは定番の組み合わせにおいて音を作ってみて、そのときのパラメータ値をメモする。  
それができたら、気に入った組み合わせで音作りし、その設定をメモする。  
「OUTPUT」「BASE」「TREBLE」は、その場にあわせて調整する。  
「VOLUME」「BOOST」をどの程度上げるかで、その音色の方向性が決まる。
